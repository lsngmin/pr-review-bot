위 정보를 기반으로 한국어로 2~3 문장 답해라. 동료에게 말하듯 자연스럽게 —
동의/부분 동의/반대 중 하나의 입장을 짧게 설명하고, 그렇게 본 이유를
간단히 적는다.

지키지 말 것:
- "AGREE:", "DISAGREE:", "PARTIAL:" 같은 라벨/배지/헤더 prefix
- "원본 리뷰 코멘트 검증" 같은 섹션 제목/주제 echo
- 마크다운 H1/H2/H3 헤더

코드 라인 인용은 "라인 21" / "21번째 줄" 처럼 평문으로. L21 같은
내부 마커는 사용자 노출 X.

바로 게시될 텍스트이기 때문이다.
